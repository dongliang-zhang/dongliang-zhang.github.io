Mac上制作OS X U盘启动

发表于 2019-10-16 分类于 器(tool) 评论数： 阅读次数：

制作 U 盘启动盘

  1. 安装软件Disk Drill for macOS.

    这个工具在国外十分有名, 制作精良, 操作友好. 如果只是制作启动盘, 使用免费的基础版即可.

  2. 下载镜像，我的百度云盘和 OneDrive 中保存了USB_Install OS X Yosemite 10.10.5.dmg镜像,这个版本相对新版本较小，只有 5 个多 G

  3. 下载后双击 dmg 文件，桌面上会出现如下图标。

  4. 打开 Disk Drill App，操作如下图


    选择你准备好的 U 盘，最好 16G，它会自动格式化划你的 U 盘，然后安装 macOS Installer 到你的 U 盘，等待完成即可。

安装系统

  1. 点开机按钮，然后一直按 option/alt，直到看到 U 盘启动。

  2. 点击安装，会进入 OS X 引导安装界面。

  3. 建议首先用磁盘工具格式化硬盘，不建议分区，一个磁盘就够了。

  4. 安装过程中可能碰到如下截图错误，提示”应用程序副本不能验证。它在下载过程中可能已遭破坏或篡改”，是因为 Apple 更新开发者证书后导致的问题。

    解决办法是是把时间改为 2015 年的一个时间,打开实用工具选择终端输入下面指令并回车
    date 032208102015.20

  5. 然后回自动安装自动重启，等待完成即可。
